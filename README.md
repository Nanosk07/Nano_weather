# Nano_weather
1️⃣ 所用API：和风天气免费API，免费版不支持港澳台查询，所以无法提供服务。

2️⃣ 界面参考：就看天气

3️⃣ 图标来源：阿里巴巴矢量图库

4️⃣ 所用开源库：ButterKnife、litepal、Glide、OKHttp
